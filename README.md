# TIENDA UFC
Este es un proyecto de ecommerce basico que se ultiliza para el curso de React dentro de CoderHause. La aplicacion permite ver una lista de productos de "UFC" y realizar compras en linea. Los productos y las ordenes de compra se almacenan en Firebase

## Tecnologias Utilizadas:
- Vite
- React
- React-Router-DOM
- Firebase
- Toastify
- Bootstrap 5.3.8

## Funcionalidades:
- Ver una lista de productos disponibles por stock
- Agregar productos al carrito de compras
- Ver el carrito de compras para chequearlo
- Realizar una orden de compra con los datos del usuario.
